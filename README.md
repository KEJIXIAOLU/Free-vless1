## CloudFlare 部署免费节点

### 下载源文件
下载作者的 Worker 文件：点击下载 [worker.zip](https://raw.githubusercontent.com/cmliu/edgetunnel/main/worker.zip)

### Pages 部署 VLESS
来到 Cloudflare 首页，点击 Workers 和 Pages ，点击 Pages  ，点击 使用直接上传创建 – 上传资产。
